
### Установить 7zip

Debian/Ubuntu:
```bash
sudo apt update && sudo apt install 7zip
```
Arch:
```bash
sudo pacman -S 7zip
```


### Запаковать в архив с паролем

Если пароль со сложными символами, обернуть в ковычки:
```bash
7z a -p"pa$$word @|" -mhe=on secure.7z *
```

```
7z        : name and path of 7-Zip executable
a         : add to archive
secure.7z : name of destination archive
*         : add all files from current directory to destination archive
-pSECRET  : specify the password "SECRET"
```

###  Распаковать архив

```bash
7z x secure.7z -p"pa$$word @|"
```
