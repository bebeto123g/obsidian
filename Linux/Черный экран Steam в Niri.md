


###### 1.  Найдите запись в Steam Desktop: Файл обычно называется steam.desktop и расположен в:

```bash
~/.local/share/applications/ (если у вас есть локальная копия)

or

/usr/share/applications/ (для системного стандарта)
```

###### 2.  Редактировать файл:

```bash
vim ~/.local/share/applications/steam.desktop

or

sudo vim /usr/share/applications/steam.desktop
```

###### 3.  Модифицировать Exec:

Оригинальная линия может выглядеть так: 
```bash
Exec=/usr/bin/steam %U
```

Вам стоит изменить это на следующее:
```bash
Exec=/usr/bin/steam -system-composer %U
```

###### 4.   Если их несколько Exec Линии (для разных окружений или действий), возможно, придётся применить аргумент ко всем или хотя бы к основному.

###### 5.  Сохранить и закрыть: Сохраните файл и закройте редактор текста.

###### 6.  Перезапустить Steam: Полностью выйдите из Steam и запустите его снова, используя свой обычный способ (например, иконкой запуска приложения или иконкой панели задач).
